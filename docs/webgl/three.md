# Three.js

## 三要素

### 场景（Scene）

场景是`Three.js`中用于组织和管理 3D 对象的容器。每个场景都可以包含多个对象，例如模型、几何体、材质等。场景可以用于组织和管理场景中的对象，例如添加、删除、遍历等操作。

### 相机（Camera）

相机是 Three.js 中用于定义场景中可见区域的对象。相机可以是透视相机或正交相机，用于模拟人眼或相机的视角。相机的位置和方向决定了场景中可见的区域。

### 渲染器（Renderer）

渲染器是 Three.js 中用于将场景渲染到屏幕上的对象。渲染器可以是 WebGL 渲染器或 Canvas 渲染器，用于在浏览器中显示场景。渲染器的配置参数可以用于调整渲染效果，例如抗锯齿、阴影等。




### 对象（Object）

对象是 Three.js 中用于表示 3D 模型、几何体、材质等的对象。对象可以是静态的，也可以是动态的，例如动画模型、交互对象等。对象的属性和方法可以用于操作和管理对象，例如位置、旋转、缩放、材质等。
